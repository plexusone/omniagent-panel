# v0.1.0

Released: 2026-07-13

Initial release of OmniAgent Panel - a multi-agent panel discussion system powered by LiveKit, LLMs, and optional AI avatars.

## Highlights

- Multi-agent panel discussions with dynamic turn-taking
- Human moderator and fully automated AI moderator modes
- HeyGen avatar integration for visual representation
- Kubernetes-ready with health endpoints and Dockerfile

## Features

### Panel Discussion System

- Coordinator orchestrates turn-taking between panelists
- Relevance-based speaking order using LLM ranking
- Shared transcript for context-aware responses
- Configurable panel size (2-4 panelists)

### Dual Moderator Modes

- **Human Mode**: Join via LiveKit Meet to moderate live
- **Auto Mode**: AI moderator generates questions and guides discussion

### Audio & Voice

- LiveKit integration for real-time audio streaming
- Multi-provider TTS via omnivoice (OpenAI, ElevenLabs, etc.)
- Deepgram STT for human mode transcription

### AI Avatars

- HeyGen LiveAvatar integration
- Per-participant avatar configuration
- Sandbox mode for development

### LLM Support

- Anthropic Claude (claude-sonnet-4-20250514)
- OpenAI GPT models
- Configurable via environment variables

### Recording

- LiveKit Egress integration
- MP4 output with configurable layout
- S3 upload support

### Deployment

- Multi-stage Dockerfile with opus support
- HTTP health endpoints (`/health`, `/ready`)
- Kubernetes deployment via agentkit

## Dependencies

- omni-livekit v0.x.x - LiveKit agent framework
- omnivoice v0.x.x - Multi-provider TTS/STT
- agentkit v0.7.0 - Deployment framework
- agentkit-k8s-pulumi v0.1.0 - Kubernetes provider
